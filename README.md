# 5月ハッカソン技育CAMP2022　チーム：beaker24

ファニープレゼントアドバイザー

## 概要

友達にあげるプレゼントを悩んでしまうことはよくあります。
そんな時に役に立てるクスッとくるようなプレゼントを選ぶお助けをするWebサイトです。

ぜひ、[こちら](https://share.streamlit.io/hiro6260000/beaker24/main/app.py)から使用してみてください！！
